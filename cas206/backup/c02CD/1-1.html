Welcome to my first web page for this class.

I am creating a web page by typing the html tags and the content in a text editor. Remember three tools are needed to create web pages: text editor, browser, and web server.

An html document is made of three main tags: html, head, and body. The html is the highest level of tag. The page begins and ends with the html tags. The head tags enclose the title of the page. The page title shows in the title bar of the browser. The body tags are the next highest level of hierarchy in a web page. The content of the page goes between the body tags.
